# Fail2Ban Module

This module automatically installs and configures the [Fail2ban](https://www.fail2ban.org)
software. By default we only monitor the SSH log files for malicious activity.

## Requirements

 * Ubuntu Linux 16.04

## Usage

```bash
$ install-fail2ban
```
